# Phở Gà Ta Đà Nẵng - Landing Page

Dự án Landing Page tĩnh cho nhà hàng "Phở Gà Ta Chính Gốc Hà Nội" tại Đà Nẵng, được xây dựng bằng Vite (Vanilla JS). Dự án sử dụng thiết kế hiện đại, giao diện responsive, với tông màu trắng kem - xanh dương nhạt (gợi nhớ biển Đà Nẵng) và đỏ gạch (ấm áp của tô phở).

## Tính Năng Bổi Bật
- **Giao diện WOW (Hero/Menu)**: Hiệu ứng cuộn mượt mà (smooth scrolling), hiệu ứng hiện (fade-in, bounce-in) sinh động.
- **Responsive Hoàn Hảo**: Hoạt động mượt mà trên cả giao diện điện thoại di động và máy tính bàn.
- **Thực Đơn (Menu)**: 10 món ăn đặc trưng với hình ảnh và giá cả.
- **Đặt Bàn/Đặt Hàng**: Biểu mẫu đặt hàng hiển thị trạng thái log và alert khi gửi thông tin.

## Công Nghệ
- HTML5 / CSS3 (Vanilla)
- ES6 JavaScript (Vanilla)
- Vite (công cụ build, máy chủ dev)

## Hướng Dẫn Cài Đặt & Chạy (Local)

1. Đảm bảo máy tính của bạn đã cài đặt [Node.js](https://nodejs.org/).
2. Mở terminal/command prompt tại thư mục của dự án (`web-quan-an`).
3. Chạy lệnh để cài đặt các thư viện (phụ thuộc Vite):
   ```bash
   npm install
   ```
4. Chạy máy chủ phát triển (Dev server):
   ```bash
   npm run dev
   ```
5. Mở trình duyệt và truy cập vào đường dẫn được hiển thị trên Terminal (ví dụ: `http://localhost:5173`).
